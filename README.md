# Student Record Management System (Java)

## 📌 Overview
This is a console-based Java application that I built to manage student records efficiently.  
The system allows users to perform basic operations like adding, viewing, updating, searching, and deleting student details.

## 🚀 Features
- Add new student records
- View all students (automatically sorted by ID)
- Search for a student using ID
- Update existing student details
- Delete student records
- Input validation using exception handling

## 🧠 Data Structures Used
- TreeMap – used to store student records in sorted order based on student ID

## ⚡ Time Complexity
- Insert: O(log n)
- Search: O(log n)
- Delete: O(log n)

## 💻 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework

## 🎯 What I Learned
- How to implement CRUD operations in Java
- Difference between TreeMap and HashMap
- Importance of choosing the right data structure
- Handling invalid input using exception handling
